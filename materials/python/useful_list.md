# List of Useful Python Stuff
This is a list of packages and tools for working with Python in HEP.

### 1. [NumPy](https://numpy.org/doc/stable/)
NumPy is a library for working with arrays (**not** Python lists) in Python. Using this library well can speed up Python routines by orders of magnitude compared to Python lists.

### 2. [Awkward](https://awkward-array.org/doc/main/)
This library extends the functionality of NumPy to _jagged_ arrays, which do not have a completely regular structure. These occur very naturally in HEP, so Awkward is a very important tool.

### 3. [Vector](https://vector.readthedocs.io/en/stable/)
Working with all sorts of vectors.
